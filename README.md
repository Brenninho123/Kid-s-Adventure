# Kid's Adventure

**Kid's Adventure** is a 2D platformer game made with **HaxeFlixel**, where the player controls a character called "Kid" who jumps across platforms, collects items, and faces enemies.  

This project is great for anyone looking to learn how to make 2D games using **Haxe** and **HaxeFlixel**.
